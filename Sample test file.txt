Usage:
======
1. Line highlight:
/r red
/g green
/b blue
/p purple
/o orange
/y yellow
/d dark
/w white
// red
#  green

2. Area highlight:
<r r>
<g g>
etc..

3. Symbol highlight:
* = $

4. Code:
```
import pythonlib
```
